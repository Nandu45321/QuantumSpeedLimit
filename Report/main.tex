\begin{abstract}
This study report focused on understanding the Quantum Speed Limit (QSL), which defines the fundamental limit on the time required for a quantum system to evolve from one distinguishable state to another. The project explored the formulation of the Mandelstam-Tamm bound, and its application in a two-level quantum system. 
\end{abstract}
